## Google Cloud Architecture Diagramming Tool

GCPでの構成図を簡単に作れるツールだそうです。それはいい！ただどこを探しても見つからず。

途方に暮れていたところ、以下のページを見つけました。

https://www.reddit.com/r/googlecloud/comments/1gmo0s1/architecture\_diagramming\_tool\_discontinued/

これによると、現在使えなくなっているようです。使えないんかーい。Googleの発表を見つけれないのが解せないです。

他のツールを探す必要があるようです。
